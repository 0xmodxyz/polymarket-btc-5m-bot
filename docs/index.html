How It Works

The bot's 5-step loop runs every 2 milliseconds until the window closes.

1

Fetch CLOB Midpoints

Get the current Up and Down prices from Polymarket's Central Limit Order Book. Each price is between $0.01 and $0.99.

2

Check Entry Condition

If up_price + down_price < 0.90, the bot enters. This guarantees at least 10¢ profit per pair at settlement.

3

Calculate Shares

shares = math.ceil(1.0 / price) — always meets CLOB's $1 minimum order size while staying close to $1 per side.

4

Buy Up (FAK Limit)

Place a fill-or-kill limit buy for Up at the midpoint price. If it doesn't match immediately, skip the entire pair.

5

Buy Down (FAK Limit)

If Up fills, place the Down side. Hold both to settlement. No sell orders — the platform pays $1 for winning shares.
